## ⚡ PASSO-A-PASSO: ABRIR AMANHÃ

### OPÇÃO 1: Usando Terminal (Recomendado)

#### 1️⃣ Abra o Terminal/PowerShell

**Windows:**
- Pressione `Win + R`
- Digite `cmd`
- Pressione Enter

**Mac:**
- Pressione `Cmd + Space`
- Digite `Terminal`
- Pressione Enter

**Linux:**
- Pressione `Ctrl + Alt + T`

#### 2️⃣ Vá para a pasta do projeto

**Copie e cole exatamente isso:**

```bash
cd /home/finaa/repos/MGOS-AIOS
```

(Funciona em Windows, Mac e Linux)

#### 3️⃣ Rode o servidor

**Copie e cole:**

```bash
npm run dev
```

#### 4️⃣ Espere aparecer:

```
➜  Local:   http://localhost:3000
```

Quando você vir isso, **o servidor está funcionando!** ✅

#### 5️⃣ Abra no navegador

Abra seu navegador (Chrome, Safari, Firefox) e digite:

```
http://localhost:3000/marketplace
```

### ✅ PRONTO! O sistema está rodando!

---

## 🆘 SE NÃO FUNCIONAR

### Erro: "npm: command not found"

**Solução:** Node.js não está instalado
- [Baixar Node.js](https://nodejs.org/)
- Instalar
- Fechar terminal e abrir novamente
- Tentar de novo

### Erro: "ENOENT: no such file or directory"

**Solução:** Você copiou errado o caminho
- Copie exatamente: `/home/finaa/repos/MGOS-AIOS`
- Não mude nada
- Tente de novo

### Erro: "Port 3000 already in use"

**Solução:** Outro programa está usando a porta
- Feche o terminal (`Ctrl + C`)
- Feche o navegador
- Tente de novo
