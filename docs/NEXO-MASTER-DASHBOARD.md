### Issue: "NEXO Master section not showing"
**Cause:** Page not loaded or cache issue
**Fix:**
1. Refresh browser (Ctrl+F5)
2. Clear cookies for site
3. Try incognito/private mode

### Issue: "Ativar NEXO button greyed out"
**Cause:** Not admin/head role or already active
**Fix:**
1. If already active: You'll see "ATIVO" status
2. If not authorized: Ask admin to elevate role
3. Check browser console for errors

### Issue: "Status cards showing 0/0"
**Cause:** NEXO not activated yet or no agents running
**Fix:** Click "Ativar NEXO" button to start

### Issue: "Taxa Sucesso stuck at 0%"
**Cause:** No tasks completed yet (they're processing)
**Fix:** Wait 5-10 minutes for tasks to complete, then refresh

### Issue: "SaÃºde showing POOR"
