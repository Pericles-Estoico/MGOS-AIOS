## Template Notes

**Using This Template:**

1. **Copy this template** for each new product listing
2. **Fill in all sections** completely before launch
3. **Adapt for channel** - remove/modify channel-specific sections as needed
4. **Use checklist items** to verify completion
5. **Track performance metrics** post-launch
6. **Update notes** with actual vs. target data
7. **Plan optimizations** based on performance

**Channel Adaptation:**
- Not every section applies to every channel
- Delete sections specific to other channels
- Adjust character limits per channel
- Modify attribute list based on category
- Customize tag count (platform limits vary)

**Template Versioning:**
- Save dated versions of successful listings
- Use high-performing listings as templates
- Document what worked in "Revision History"
- Scale proven templates to similar products
